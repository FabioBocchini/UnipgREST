Per eseguire sia il servizio Rest che il Client è necessario installare Node.js => https://nodejs.org/it/

Una volta installato aprire il cmd, navigare fino alla cartella REST e Client ed eseguire il comando 'npm install' su entrambe
Una volta installati basta eseguire 'npm start' per eseguirli, la pagina web è disponibile all'indirizzo: http://localhost:3000
il server sarà di default in ascolto sulla porta 3001 

per caricare il database installare pgAdmin, aprire il cmd, navigare fino alla cartella PostgreSQL/'versione'/bin ed eseguire il comando 'psql -U username dbname < database.pgsql'
